Shared Hardcore Survival MOD

A multiplayer shared hardcore survival mod for Minecraft Forge 1.20.1.

Features
1. Player Waiting System
Waits for 4 players to join.
Displays the current player count on the HUD (0/4 to 4/4).
Records each player's username and UUID.
2. Game Start System
Randomly assigns each of the four players to one of the four corners of the world.
Each player is placed at a unique location.
Automatically teleports players to their assigned positions.
The challenge begins once all four players have arrived.
3. Hardcore System
If any player dies, the entire team dies instantly.
The challenge ends immediately.
4. Shared Inventory
All four players share a single inventory.
Adding, removing, or viewing items is synchronized in real time.
5. Shared Chests
All chest inventories are fully synchronized.
Any changes made by one player are instantly reflected for everyone else.
6. Shared Health
Damage taken by any player is applied to all players.
Healing received by any player is shared with the entire team.
7. Shared Hunger
Hunger level is synchronized.
Saturation level is synchronized.
When one player eats, all players receive the same hunger and saturation restoration.
8. HUD
Displays the current waiting player count.
Displays the survival timer.
Displays challenge completion or failure information.
9. Commands

/shared start    - Start the challenge
/shared reset    - Reset the game
/shared end      - End the challenge
/shared info     - Display game information
/shared bag      - Open the shared inventory

Not Shared

The following inventories and abilities remain individual:

Sophisticated Backpacks
Armor slots
Off-hand slot
Curios slots
Enigmatic Legacy abilities
